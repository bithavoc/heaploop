Synchronous, Concurrent, Non-Blocking Application Framework for D

dub install heaploop

Concurrency Model

Activate Fibered-Concurrency for any event just by switching subscription operators:

Non-Concurrent Application:

server.listen ^= (client){
    // executes in the same calling fiber.
};

Concurrent Application:

server.listen ^^= (client){
    // executes in a new fiber.
};

Example TCP Server:

import heaploop.looping;
import heaploop.networking.tcp;

void main() {
    loop ^^= {
        auto server = new TcpStream;
        server.bind(3000);
        server.listen ^^= {
            client.write(cast(ubyte[])”Hello World”);
        };
    };
}

Under the Hood

Heaploop is built on top of libuv's high-perfomance and cross-platform capabilities. The rest is bare-metal perfomance, like it’s supposed to be.
